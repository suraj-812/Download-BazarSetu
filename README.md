# BazarSetu APK Download Page

A simple, modern, and responsive HTML page designed for direct APK downloads. This project provides a premium user experience for Android users to download and install the BazarSetu application.

## Features
- **Modern UI**: Clean black & white theme with premium typography (Inter).
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop devices.
- **Micro-animations**: Smooth entrance animations and hover effects.
- **Direct Download**: Optimized anchor tag implementation with the `download` attribute.
- **Lightweight**: Zero external dependencies (except Google Fonts).

## Project Structure
- `index.html`: The main landing page.
- `bazarsetu.apk`: The Android application package file.

## How to Deploy
1. Host these files on any static web server (GitHub Pages, Vercel, Netlify, etc.).
2. Ensure `bazarsetu.apk` remains in the same directory as `index.html`.
3. Update the version and file size in `index.html` as needed.
